# Zugprojekt

## Einmalige Einrichtung

**Voraussetzung:** [Node.js](https://nodejs.org/) (LTS) installieren.

Dann im Projektordner:

```bash
npm install
```

Damit werden alle Abhängigkeiten inkl. Firebase SDK und Firebase CLI installiert. Anschließend:

- **App starten:** `npm run dev`
- **Firebase (z. B. Login/Deploy):** `npx firebase login` bzw. `npx firebase deploy`

Alternativ alles in einem Schritt: `npm run setup`
